
# Task 1 - Basic Network Scanning with Nmap

## Objective
Perform a network scan using Nmap to identify open ports and services on the local machine.

## Tools Used
- Nmap
- Zenmap
- Windows 11

## Steps
1. Installed Nmap.
2. Opened Zenmap.
3. Scanned the target 127.0.0.1.
4. Analyzed the scan results.

## Outcome
Successfully performed a basic network scan and identified the available network ports and services.
