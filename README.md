
# Task 7 - Vulnerability Scanning with Nikto

## Target
DVWA (Damn Vulnerable Web Application)

## Scan URL
http://10.145.23.140/dvwa/

## Tool Used
Nikto v2.1.5

## Scan Summary
- Host tested: 1
- Items checked: 6544
- Errors: 0
- Reported findings: 35

## Vulnerabilities Identified

1. Directory Indexing Enabled
- Directory listing is enabled, which may expose files and folders.

2. Missing X-Frame-Options Header
- Missing security header may allow clickjacking attacks.

3. HTTP TRACE Method Enabled
- TRACE method can increase Cross-Site Tracing (XST) risk.

4. Server Information Disclosure
- Apache and PHP version details are visible.

## Remediation

- Disable directory indexing.
- Add security headers.
- Disable HTTP TRACE method.
- Hide server version information.
- Restrict unnecessary HTTP methods.

## Tools Used
- Nikto
- Ubuntu WSL
