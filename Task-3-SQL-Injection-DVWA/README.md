# Task 3: SQL Injection on DVWA

## Objective
Demonstrate a SQL Injection vulnerability by exploiting the DVWA login form.

## Tools Used
- DVWA (Damn Vulnerable Web Application)
- XAMPP
- Web Browser

## Security Level
Low

## Steps Performed
1. Opened DVWA SQL Injection module.
2. Changed security level to Low.
3. Entered SQL Injection payload in the User ID field.
4. Observed the application response.

## Payload Used
1' OR '1'='1

## Result
SQL Injection vulnerability was successfully demonstrated.
